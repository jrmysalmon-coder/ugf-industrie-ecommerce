## Vue d'ensemble

UGF Industrie est une plateforme e-commerce **B2B** specialisee dans la vente de pieces industrielles et equipements. Elle est destinee aux professionnels (entreprises, artisans) avec un systeme de validation de compte.

### Objectifs architecturaux

- **Performance**: Temps de reponse API < 200ms (p95)
- **Disponibilite**: Uptime cible 99.9%
- **Securite**: Authentification JWT, HTTPS uniquement, validation des donnees
- **Maintenabilite**: Code typise, tests automatises, documentation a jour

                                                                                                                                                                                                                                                                                            ## API REST
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            ### Conventions
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            - **Base URL**: `/api/v1`
                                                                                                                                                                                                                                                                                            - **Format**: JSON (`Content-Type: application/json`)
                                                                                                                                                                                                                                                                                            - **Auth**: JWT Bearer token dans le header `Authorization`
                                                                                                                                                                                                                                                                                            - **Pagination**: `?page=1&limit=20`
                                                                                                                                                                                                                                                                                            - **Filtres**: `?category=hydraulique&brand=smc`
                                                                                                                                                                                                                                                                                            - **Tri**: `?sort=price_ht&order=asc`
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            ### Endpoints principaux
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            ```
                                                                                                                                                                                                                                                                                            # Auth
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/register      - Inscription
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/login         - Connexion
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/logout        - Deconnexion
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/refresh       - Refresh token
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/forgot-password
                                                                                                                                                                                                                                                                                            POST   /api/v1/auth/reset-password
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            # Produits
                                                                                                                                                                                                                                                                                            GET    /api/v1/products           - Liste (filtre, tri, pagination)
                                                                                                                                                                                                                                                                                            GET    /api/v1/products/:slug     - Detail produit
                                                                                                                                                                                                                                                                                            GET    /api/v1/products/search    - Recherche full-text
                                                                                                                                                                                                                                                                                            GET    /api/v1/categories         - Liste categories
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            # Commandes (auth requise)
                                                                                                                                                                                                                                                                                            GET    /api/v1/orders             - Mes commandes
                                                                                                                                                                                                                                                                                            POST   /api/v1/orders             - Creer commande
                                                                                                                                                                                                                                                                                            GET    /api/v1/orders/:id         - Detail commande
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            # Devis (auth requise)
                                                                                                                                                                                                                                                                                            GET    /api/v1/quotes             - Mes devis
                                                                                                                                                                                                                                                                                            POST   /api/v1/quotes             - Creer devis
                                                                                                                                                                                                                                                                                            PATCH  /api/v1/quotes/:id         - Modifier devis
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            # Admin (role admin requis)
                                                                                                                                                                                                                                                                                            GET    /api/v1/admin/users        - Gestion clients
                                                                                                                                                                                                                                                                                            PATCH  /api/v1/admin/users/:id    - Valider/suspendre
                                                                                                                                                                                                                                                                                            GET    /api/v1/admin/orders       - Toutes les commandes
                                                                                                                                                                                                                                                                                            PATCH  /api/v1/admin/orders/:id   - Mettre a jour statut
                                                                                                                                                                                                                                                                                            GET    /api/v1/admin/products     - Gestion catalogue
                                                                                                                                                                                                                                                                                            POST   /api/v1/admin/products     - Ajouter produit
                                                                                                                                                                                                                                                                                            PUT    /api/v1/admin/products/:id - Modifier produit

                                                                                                                                                                                                                                                                                                                          ## Authentification
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ### Flux JWT
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                          1. POST /auth/login -> { accessToken, refreshToken }
                                                                                                                                                                                                                                                                                                                          2. accessToken: expire 15min, stocke en memoire
                                                                                                                                                                                                                                                                                                                          3. refreshToken: expire 7j, stocke en httpOnly cookie
                                                                                                                                                                                                                                                                                                                          4. Refresh automatique via /auth/refresh
                                                                                                                                                                                                                                                                                                                          5. Logout: invalide le refreshToken en BDD
                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ### Middleware auth
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          Le fichier `src/middleware.ts` protege:
                                                                                                                                                                                                                                                                                                                          - `/compte/**` - Espace client
                                                                                                                                                                                                                                                                                                                          - `/admin/**` - Interface administration
                                                                                                                                                                                                                                                                                                                          - `/api/v1/orders/**` - Commandes
                                                                                                                                                                                                                                                                                                                          - `/api/v1/quotes/**` - Devis
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ### Roles
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          | Role | Acces |
                                                                                                                                                                                                                                                                                                                          |------|-------|
                                                                                                                                                                                                                                                                                                                          | `guest` | Catalogue public uniquement |
                                                                                                                                                                                                                                                                                                                          | `client_pending` | Catalogue + devis (pas de commande) |
                                                                                                                                                                                                                                                                                                                          | `client_active` | Tout (catalogue + commande + devis) |
                                                                                                                                                                                                                                                                                                                          | `admin` | Tout + back-office |
                                                                                                                                                                                                                                                                                                                          | `superadmin` | Tout + gestion admins |

                                                                                                                                                                                                                                                                                                                          ## Paiements Stripe
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ### Flux Checkout
                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                          1. Client valide le panier
                                                                                                                                                                                                                                                                                                                          2. POST /api/v1/stripe/create-session
                                                                                                                                                                                                                                                                                                                             -> Cree une Stripe Checkout Session
                                                                                                                                                                                                                                                                                                                             3. Redirection vers stripe.com
                                                                                                                                                                                                                                                                                                                             4. Paiement effectue
                                                                                                                                                                                                                                                                                                                             5. Stripe appelle le webhook /api/v1/stripe/webhook
                                                                                                                                                                                                                                                                                                                             6. Verification signature webhook (STRIPE_WEBHOOK_SECRET)
                                                                                                                                                                                                                                                                                                                             7. Evenement checkout.session.completed:
                                                                                                                                                                                                                                                                                                                                - Cree la commande en BDD
                                                                                                                                                                                                                                                                                                                                   - Reduit le stock
                                                                                                                                                                                                                                                                                                                                      - Envoie l'email de confirmation
                                                                                                                                                                                                                                                                                                                                      8. Client redirige vers /commande/confirmation
                                                                                                                                                                                                                                                                                                                                      ```
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      ### Evenements Stripe ecoutas
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      - `checkout.session.completed` - Paiement confirme
                                                                                                                                                                                                                                                                                                                                      - `payment_intent.payment_failed` - Echec paiement
                                                                                                                                                                                                                                                                                                                                      - `customer.subscription.deleted` - (futur: abonnements)

                                                                                                                                                                                                                                                                                                                                      ## Securite
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      ### Mesures implementees
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      - **HTTPS**: Force sur tous les environnements
                                                                                                                                                                                                                                                                                                                                      - **Helmet.js**: Headers HTTP securises (CSP, HSTS, etc.)
                                                                                                                                                                                                                                                                                                                                      - **Rate limiting**: 100 req/15min (global), 10/15min (auth)
                                                                                                                                                                                                                                                                                                                                      - **Input validation**: Zod pour tous les inputs API
                                                                                                                                                                                                                                                                                                                                      - **SQL Injection**: Requetes parametrees (Prisma/pg)
                                                                                                                                                                                                                                                                                                                                      - **XSS**: Echappement auto React + Helmet CSP
                                                                                                                                                                                                                                                                                                                                      - **CSRF**: SameSite cookies + CORS strict
                                                                                                                                                                                                                                                                                                                                      - **Secrets**: Variables d'env uniquement, jamais en code
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      ### Audit log
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      Toutes les actions sensibles sont loggees:
                                                                                                                                                                                                                                                                                                                                      - Connexions/deconnexions
                                                                                                                                                                                                                                                                                                                                      - Modifications de compte
                                                                                                                                                                                                                                                                                                                                      - Changements de statut commande
                                                                                                                                                                                                                                                                                                                                      - Actions admin
